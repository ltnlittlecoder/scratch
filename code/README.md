# Scratch闪电 在线编程平台
=============================================

### 介绍
    scratch闪电 是一个轻量级的Scratch在线编程、分享平台

### 功能模块：
1. 注册登录；
2. Scratch创作；
3. 作品管理；
4. 用户管理；
5. 个人信息管理；
6. 作品播放、点赞、收藏、分享。
